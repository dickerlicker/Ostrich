Configuring Ostrich VPN

If you are using your own Kali Image:
Install OpenVPN on your machine:
sudo apt-get install openvpn network-manager-openvpn network-manager-openvpn-gnome

Download our Kali Image here.

Steps to Configure OpenVPN:
On your Kali Linux:

  1. Open Wired Settings
     
  2. Add a new VPN Connection
     
  3. Click Import from File...
     
  4. Select the .ovpn file you downloaded from the question
  5. Once done, you should see something like this, ensure that the gateway is correct, then enter the username and password as specified in the question.
     
  6. Click Add
     
  7. Lastly, to connect, click on the connect button
     
  8. Once connected, you should see something like this:
     

Troubleshooting:
If you see something like this:
 
Check the following:

  • Ensure that you have internet connection by pinging Google.com
  • Ensure that the VPN Gateway as shown in Step 5 is correct.
  • Ensure that you have used the correct .ovpn file.
